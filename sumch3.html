CHAPITRE 03: LE MODELE OSI


1)- Les composants matériel d'un réseau informatique:

a-les cartes réseau:

Sont des composants matériels qui permettent à un ordinateur de se connecter à un réseau,
sont installées à l'intérieur de l'ordinateur et sont généralement connectées à une prise Ethernet ou Wi-Fi,
les cartes réseau permettent à l'ordinateur de transmettre et de recevoir des données via des protocoles de communication spécifiques.

b- Switch:

Commutateur réseau est un composant matérièl utilisé pour connecter des appareils sur un réseau local.
Le switch utilise une table de correspondance et l'utilise pour envoyer les données uniquement à l'appareil destinataire.

c- Hub:

Concentrateur est un composant matérièl utilisé pour connecter plusieurs appareil sur un réseau local,
le HUB envoie les paquets au tous les noeuds connectes au HUB.

2)- Le modèle OSI:

le modèle OSI est utilisé généralement comme références aux différentes couches.
Open Systems Interconnection est un modèle de référence pour les communications de réseau qui divise
les communications en sept couches sont les suivants: La couche physique, la couche liaison de données,
la couche réseau, la couche transport, la couche session, la couche présentation, la couche appliction.

3)- Le modèle TCP/IP:

Le modèle Transmission Control Protocol/Internet Protocol est un modèle de référence pour
les communication de réseau.
Le modèle TCP/IP comport quatre couches:
-couche application
-couche transport (TCP)
-couche internet (IP)
-couche accès réseau.
TCP/IP est le modèle de référence pour les protocoles de communication de réseau utilisés pour la conception
et la mise en oeuvre des réseaux d'entreprise, des réseaux Internet et des réseaux sans fil.

4) L’adressage :

Les communications dans un réseau obéissent à des règles :
• Adressage: qui permet d'identifier de manière unique les deux unités en
communication
• Architecture: qui définit les rôles endossés par les deux unités
• Protocoles: qui assurent l'échange des données

un adressage IP :
Une adresse IP (Internet Protocol) est le numéro qui identifie chaque ordinateur dans les réseaux
qu’il appartient. Il y’a deux mode d’adressage IPV4 et IPV6 :

- ADRESSE IPV4:
Chaque paquet contient : adresse source sur (32 bits) et une adresse destinataire sur (32bits)
- ADRESSE IPV6 :
Les adresses sont codées sur 128 bits. La représentation textuelle se fait en découpant le mot de 128 bits de l'adresse en 8 mots de 16 bits séparés par le caractère «:», chacun d'eux est en hexadécimal.

5) L’adresse IPV4:

Les 4 octets constituent deux parties, une partie pour Partie réseau et l'autre pour la partie hôte.
Le nombre des bits contenus dans la partie hôte détermine le nombre des machines possible sur ce réseau.
Les différents types d’adresse IPV4: l’adressage IPV4 s’effectué selon les types suivants :
L’adresse réseau : tous les bits de la partie hôtes=0 exemple:( 10.0.0.0) ,c'est-à-dire tous les
hôtes(machines) de ce réseau ont la même partie réseau.

L’adresse de diffusion :Cette adresse permet à un pc d'envoyer des paquets à l'adresse de diffusion de réseau,
et tous les hôtes reçoivent ces données, sans distribuer une paquet pour chaque pc.

L’adresse hôtes :On attribue les valeurs situés entre l’adresse réseau et l’adresse de
diffusion sur {machines} situé sur cette réseau.
