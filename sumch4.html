CHAP04: Internet et le web

1-Intro :

Le World Wide Web est l'une des applications du support de communication général autrement dit internet qui a été inventé par Tim Berners-Lee et Robert Caillau. Elle sert notamment à
consulter des pages accessibles sur des sites comme des documents textuels, à servir des ressources de différentes natures reliés par un lien hypertexte.Ce dernier permet de
suivre une URL(Uniform Resource Locator) qui est le nom unique de la page et contient :

• L'adresse du serveur.

• L'adresse de la page sur le serveur.

2-Aspects techniques du web :

Les 3 méchanismes du web se résument à :

1° URL :

--> L'accés à plusieurs ressources grace au web.

--> Le mécanisme de localisation peut faire appel au protocole DNS.

2° HTTP :

--> Protocole de niveau applicatif.

--> Paradigne client serveur.

--> Protocole sans états.

3° HTML :

--> Lanage de description de page web.

--> Textes, images et autres objets.

--> Lien hypermedias entre les pages.

--> Programmation déclarative.


3-Page web et site web:

Une page web est:

--> Un document multimédia.

--> est décrit par un langage de description.

--> Stockée dans un fichier.

--> Localisée sur internet grace à une URL .

Un site web est:

--> Un regroupement de pages web reliées par des liens hypertexte.

--> Accessible à partir d'une URL de base communes.

--> Construit sous forme d'arborecence de dossiers et de fichiers .

4-Création d'une page web HTML:

Un documment HTML est:

--> Un fichier de texte.

--> Sructuré par des balises.

--> Crée dans un éditeur de page web ou tout éditeur de texte.

--> Suivant une syntaxe pécise .

Les principes de langage HTML:

• Les quatres balises principales: qui sont utilisées pour le titre, le corps, l'entete
et les limites du document: <html >...</html>, <head>...</head>, <title>...</title>, <body>...</body>

• Pour les titres et les paragraphes:
<hn>... <hn>, <p>...<p>
